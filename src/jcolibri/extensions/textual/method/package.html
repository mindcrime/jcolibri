Textual CBR Methods.

This package stores several standard TCBR Methods. It includes:

  • Words filters
  • Part-of-Speech tagging
  • Stemmer algorithms
  • Name Extraction
  • Phrase identification
  • Glossary
  • Thesaurus (WordNet)
  • Feature Extraction
  • Topics classification
  • Basic Information Extraction

These methods uses several external packages:

  • Open Natural Language Processor - Grok: This package uses the Maximum Entropy model to perform natural language algorithms.

    Information about opennlp MaxEnt model is avaliable in: http://maxent.sourceforge.net

    OpenNLP is available in: http://opennlp.sourceforge.net.

    Really, Opennlp only defines a set of Java interfaces and implements some basic infrastructure for NLP components. This is the base of other projects/packages like Grok. Grok implements and extends those interfaces so this is the package used in these methods.

    The main advantage of Opennlp is that can be extended by TCBR developers to create new NLP algorithms and classificators.

  • SnowBall. It is a small string processing language designed for creating algorithms for use in Information Retrieval.

    This package contains stemmers algorithms for English, Russian, French, Spanish, Portuguese, Italian, German, Dutch, Swedich, Norwegian, Dannish and Finnish.

    Snowball is available in: http://snowball.tartarus.org

  • Java WordNet Library. It is a Java API for accessing the WordNet relational dictionary.

    This package is available in: http://sourceforge.net/projects/jwordnet
